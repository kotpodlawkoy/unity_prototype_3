# Проведённая работа
В контексте данного проекта были реализованы:
* Прототип **side-view** 3D игры, где надо перепрыгивать через препятствия (по типу 3D версии динозаврика из хрома когда инета нет :( ), где была проведена работа с:
  * Физикой RigidBody
  * Более проработанной механикой прыжков и работой
  * Более тонкой работой с аудиосистемой и проигрыванием нужных в данный момент event-based звуков
  * Работа с готовыми VFX системы частиц
  * Работа с готовыми анимациями персонажа
  * Работа в Unity Animator

* Прототип **side-view** 3D игры (путь: ./Challenge 3/), где надо играя за шарик ловить монетки и избегать БОМБ!!!
  * Реализованы аккуратные верхние и нижние границы полёта шарика с использованием физического движка
  * Также проведена работа с системой частиц и аудиосистемой, как в предыдущем прототипе
